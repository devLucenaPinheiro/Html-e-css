Usando links externos

Você pode acessar o meu repositório publico no GitHub

Neste parágrafo colocamos um link externo no site utilizando a tag <a> que é a tag de ancora(link),usamos href="link do site externo" para acessar meu repositório no GitHub(sempre devemos colocar o link do site que desejamos ir no click dentro de href que seria a referência do proximo link), tambem utilizamos o target="_blank" para abrir o link em uma nova página sem fechar o nosso síte, e utilizamos o rel="external" para indicar que o link é externo.

Usando links internos

Aqui podemos linkar uma página interna que esta no próprio código.
