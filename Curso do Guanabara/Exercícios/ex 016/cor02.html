Testando Gradiante

Neste exercicio utilizamos a tag <style> para criar um CSS interno, no CSS, utilizamos o background-attachment: fixed; para deixar o fundo responsivo de acordo com os scrolls do mouse, e para criar o degrade no fundo utilizamos o background-image: linear-gradient em 45 graus(utiliza-se deg) junto com um conjunto de cores para fazer este efeito(que tambem podemos mudar o quanto ela aparece usando uma % depois do código hexadecimal).
