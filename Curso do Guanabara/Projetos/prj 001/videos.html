Inserindo vídeos hospedados localmente

Este vídeo esta hospedado no meu próprio servidor.

Para fazer com que o vídeo apareça aqui usamos a tag <video>, dentro da tag vídeo podemos usar os parâmetros de width(largura), poster para thumbnail(foto de capa), controls para dar controles ao vídeo(controle de volume e segundos do vídeo) e loop para deixar o vídeo repetindo, utilizamos tambem a tag <source> com parâmetro src= para indicar a fonte local do vídeo, e depois utilizamos type para indicar qual o formato do vídeo(mp4,webm e m4v)
