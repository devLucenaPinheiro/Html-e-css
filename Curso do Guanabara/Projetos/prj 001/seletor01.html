Aprendendo ID no CSS, ID seria para identificar um seletor específico, como por exemplo identificar o h1 no topo do site como principal e apenas modificando ele e não todos os outros h1s para fazer isso usamos o id dentro da tag desejada para especifica-la e depois na CSS quando formos escrever o seletor utilizamos "#" e depois o id,

Sempre que colocar id em HTML = # em CSS

Sempre que colocar class em HTML = .em CSS

o id você pode usar apenas um por elemento, o class serve para você poder usar em mais de um, você pode colocar 2 class em um elemento ao mesmo tempo apenas os separando por espaço sem a necessidade de utilizar virgula. e você pode colocar id e class juntos, mas se os 2 usam uma configuração parecida o id sobrepõe o class.
